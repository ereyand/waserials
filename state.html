Reflection

As I learned about the three serial killers that operated in Washington state, I learned about the many victims and their stories and how it might have taken a long time but they received justice. They received justice thanks to DNA Analysis and the work of forensic scientists.

Difficulties

The most difficult aspect of my project would probably be remembering the code for a website and how there are certain rules to different types of code. I utilized HTML and CSS for this project and had to create two very different codes for each as well as learning how to link them to each other.

Coding is VERY time consuming, especially if you focus on the little details as I did. Especially tables take up the most time and you have to be organized or you'll loose track of where you are at. Coding tables is difficult to understand but when do correctly they look great. My table for the victims of Gary Ridgway is more tha 200 lines of code!

Code

HTML is Hyper Text Markup Language and its basically the structure and foundation of the website. Its also where the text and images go.

CSS is Cascading Style Sheets and its the decorative aspect of coding, with CSS you can change background, font and size.
